# Technology Stack

## Frontend
- **Framework:** React
- **Build Tool:** Vite
- **Language:** JavaScript/JSX
- **Routing:** React Router
- **HTTP Client:** Axios

## Backend
- **Runtime:** Node.js
- **Framework:** Express
- **Language:** TypeScript
- **Database:** Supabase (PostgreSQL)

## Core Libraries
- **Crawling/Browser Automation:** Puppeteer
- **HTML Parsing:** Cheerio
- **CSS Selector Generation:** css-selector-generator

## Specialized Services
- **Crawling Service:** Differentiated services for:
    - **Full Site Scan:** Recursive crawling of the entire domain.
    - **Script-Based Verification:** Targeted scanning of pages that explicitly include the user's tracking script (likely for verification or limited scope audits).
